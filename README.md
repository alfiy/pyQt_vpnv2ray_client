# VPN & Proxy Client

This is a simple VPN and proxy client for Linux using PyQt.

## Requirements

- Python 3.x
- PyQt5
- OpenVPN
- V2Ray/Xray

## Setup

1. Install the required Python packages:
   ```bash
   pip install -r requirements.txt
